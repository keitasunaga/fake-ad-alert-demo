# FakeAdAlertDemo - コンセプトドキュメント

## 概要

SNS上のフェイク広告（VC未認証）を自動検出し、ユーザーに警告を表示するChrome拡張機能のデモアプリケーション。

### ターゲット
- 広告代理店（サイバーエージェント、電通、博報堂等）の役員向け営業デモ

### 関連プロジェクト
- `vericerts-zero-share-verifier`: スマホ共有機能でフェイク広告を判定するデモアプリ
- `vericerts-zero-browser-extension`: サイトのVC確認用Chrome拡張（別プロジェクト）

---

## 対象SNS

| 優先度 | SNS | 理由 | ステータス |
|--------|-----|------|-----------|
| 🥇 1st | Instagram | Meta詐欺広告問題が社会問題化、フィード型で検出しやすい | 未着手 |
| 🥈 2nd | TikTok | フィード構造がインスタに類似、若年層向け詐欺問題 | 未着手 |
| 🥉 余力 | YouTube | インストリーム広告の検出が技術的に難しい、要調査 | 保留 |

---

## 機能仕様

### 自動チェック機能
- ユーザーアクション不要で広告を自動検出
- フィード一覧・個別投稿の両方に対応
- MutationObserverでスクロール時の動的読み込みも監視

### 判定ロジック
- 広告主名のパターンマッチ方式
- YAML/JSONでホワイトリスト/ブラックリストを管理
- デモ用のモック判定（実際のVC検証は行わない）

### 表示パターン

| 状態 | 表示 | 色 |
|------|------|-----|
| フェイク広告（未認証） | ⚠️ 警告オーバーレイ | 赤系 |
| 認証済み広告 | ✅ VeriCerts認証済みバッジ | 緑 or 青 |

## UI設計方針

### コンセプト
- **SNSに溶け込むデザイン**: 各SNSのデザイン言語に合わせた自然なUI
- **ぱっと見でヤバさがわかる**: フェイク広告は即座に危険と認識できる警告

### Instagram向けUI案

```
フェイク広告の場合:
┌─────────────────────────────┐
│ [広告主アイコン] 広告主名    │
│ 広告 ・ Sponsored           │
├─────────────────────────────┤
│                             │
│    [広告画像]               │
│         ┌──────────────┐    │
│         │ ⚠️ 未認証広告 │    │  ← 半透明オーバーレイ
│         │  詐欺の可能性  │    │
│         └──────────────┘    │
│                             │
└─────────────────────────────┘

認証済み広告の場合:
┌─────────────────────────────┐
│ [アイコン] 広告主名 ✅       │  ← 名前横に認証バッジ
│ 広告 ・ VeriCerts認証済み   │
└─────────────────────────────┘
```

### TikTok向けUI案
- Instagramと同様のフィード型アプローチ
- 縦スクロールのショート動画に合わせた警告表示

## データ構造

```yaml
# config/ad-verification.yml

whitelist:  # VC付与済み（認証済み広告主）
  - name: "トヨタ自動車"
    patterns: ["トヨタ", "TOYOTA", "toyota.jp"]
  - name: "ソニー"
    patterns: ["Sony", "SONY", "ソニー"]
  - name: "Apple Japan"
    patterns: ["Apple", "apple.com"]

blacklist:  # フェイク（未認証広告主）
  - name: "怪しい投資案件"
    patterns: ["今すぐ100万円", "誰でも簡単に稼げる", "副業で月収100万"]
  - name: "なりすまし系"
    patterns: ["前澤友作", "堀江貴文", "孫正義"]
  - name: "情報商材系"
    patterns: ["限定公開", "今だけ無料", "残りわずか"]
```

## 注意事項

1. **デモ専用**: このアプリは営業デモ用であり、本番環境での使用は想定していません
2. **VC検証はモック**: 実際のVC検証は行っておらず、広告主名のパターンマッチで判定しています
3. **SNSのDOM依存**: 各SNSのDOM構造変更により動作しなくなる可能性があります
